Ce cours

Ce que ce cours n'est pas

  • Un cours de biologie
  • Un cours de maths
  • Un cours théorique de machine learning
  • Une longue suite (décousue) d'équations à apprendre par coeur

Ce qu'est ce cours

  • Un lieu d'échange
  • Un partage d'expérience de chercheur
  • Un partage d'expérience de professionnel de la data (médicale)
  • Une confrontation à la vraie vie

Comment ça va se passer pour vous

  • Retenez bien toutes les slides, y aura un exam sur table hyper compliqué
  • Vous êtes notés uniquement sur le TP
  • Le TP est très libre et vous serez très autonomes
  • Le TP vous apprend que la théorie, c'est franchement pas le plus dur (si vous ramez c'est normal)
  • Y a pas de raison de pas finir le TP (ni de pas le finir d'ailleurs)
